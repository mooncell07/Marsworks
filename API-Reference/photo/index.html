    Photo

    A class representing a Photo.

    Attributes:

    Name Type Description
    photo_id Optional[int]

    ID of the photo.
    sol Optional[int]

    Sol when the photo was taken.
    img_src str

    Image url.

    Info

    Images default to medium size for Curiosity, Opportunity & Spirit and large size for Perseverance.
